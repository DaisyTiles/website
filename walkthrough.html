Get Started

Download Build Program Next-Steps

Download and Install Software (MacOS)

  1. Visit the software page and download the the installer for MacOS
  2. The installer should have the extension .dmg. Double click on it and a window should pop up
  3. In the window, drag the application over to the application folder
  4. The app is ready to launch! Hit Command + Space and search for "Daisy Tiles"
  5. Click the app, and it should open
  6. The app will show an installation screen on first launch. If something fails, you will see an alert. If such, please open the file it does and contact me with its contents

Build

  1. In your kit, find the tile that says 'stem' (You should see a USB port on it). This is the main tile that acts as a host to all of the other tiles.
  2. You should be able to see that it has a connector on each side, this is where you will attach any other tiles (called peripherals) to.
  3. You'll see that every peripheral also has 4 sides, but notice how only one of those has an arrow going into the center of the tile. This is the input side of the tile.
  4. Go ahead and find the red LED tile, and connect it to any of the sides on the stem tile.
  5. Now grab the button tile and connect it either to the stem tile, or the LED tile.

Program

  1. Open the TileIDE and drag out the following blocks in this order.
  2. Note that the read button has been dragged inside of the set led button.
  3. Connect the stem tile to the computer with the provided cable. There should be two green dots under the program button, one of which should say Connected to Device.
  4. Click the Program button on the left, and you should see three green boxes saying BUILD: PASSBOOT: PASSFLASH: PASS in the bottom left corner. If one of them is orange and says FAIL, make sure that the device is connected. If thats not the issue, please copy and paste the log directly over the green/orange/gray boxes, and contact me.
  5. If all boxes are green, your program is ready! Try pushing, holding, and releasing the button to see how the system reacts.
